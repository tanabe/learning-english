## 英単語・熟語

* stipulate
    * (他) 〜を規定する 
* disappointing
    * (形) 期待外れの
* engaging
    * (形) 魅力のある
* look as if
    * 〜のようだ
* be instrumental in doing
    * 〜するのに役立つ
* unless otherwise notified
    * 特に通知がないかぎり
* upon
    * 〜の時に
* whether + to 不定詞
    * 〜するかどうか
* of + 抽象名詞 == 形容詞
* value at
    * 見積もる
* dedicated to doing
    * 〜することに専念する
* report to work
    * 出勤する
* applications
    * 応用、用途
* be excited about X
    * X にワクワクしている
* be committed to X
    * X に専念する
* another 数詞 X でさらに X
    * There are another three rooms upstairs.
* credit X to Y
    * X を Y のおかげとする
    * to は前置詞
* in which / at which == where
